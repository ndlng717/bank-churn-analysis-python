# Bank Customer Churn Analysis

## Project Overview
This project analyzes customer churn behavior in a bank using Python.  
The goal is to identify key factors driving customer attrition and provide actionable business insights.

---

## Dataset
- Source: Kaggle (Churn Modelling Dataset)
- Records: 10,000 customers
- Target Variable: `Exited` (1 = churn, 0 = retained)

---

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

---

## Key Analysis Performed
- Data cleaning and validation
- Exploratory data analysis (EDA)
- Churn rate analysis across multiple features

---

## Key Insights
- Inactive customers have significantly higher churn rates
- Customers in Germany show the highest churn
- Higher balance customers are more likely to churn
- Older customers tend to leave more frequently

---

## 🎯 Business Recommendations
- Improve engagement for inactive customers
- Target high-value customers with retention offers
- Investigate regional issues in high-churn markets
- Tailor strategies for older customer segments
